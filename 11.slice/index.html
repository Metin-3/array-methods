JavaScript slice() Metodu

slice() metodu JavaScript-də massivlər (və ya string obyektləri) üzərində istifadə olunur. Bu metod, mövcud massiv və ya stringdən müəyyən bir hissəni seçib, orijinal obyektə toxunmadan yeni bir massiv və ya sətir yaradır.

Sintaksis: array.slice(start, end);

  • start (istəyə bağlıdır): Hissənin başlanğıc indeksini göstərir.
  • Əgər mənfi bir rəqəmdirsə, massivdə sondan başlayaraq sayılır.
  • Əgər göstərilməsə, start dəyəri avtomatik olaraq 0 qəbul edilir.
  • end (istəyə bağlıdır): Hissənin bitmə indeksini göstərir. end indeksi daxil edilmir (yəni end-dən əvvəlki elementlər daxil edilir).
  • Əgər göstərilməsə, end dəyəri avtomatik olaraq massiv uzunluğuna (array.length) bərabər götürülür.
  • Mənfi rəqəm verilərsə, sondan başlayaraq sayılır.

Xüsusiyyətlər:

  • Orijinal massiv dəyişdirilmir (immutability).
  • Yeni massiv və ya string qaytarır.
  • end həmişə exclusive (daxil edilmir).

slice() vs splice()

  • slice(): Orijinal massivi dəyişdirmir. Yalnız yeni bir massiv yaradır.
  • splice(): Orijinal massivi dəyişdirir və silinmiş elementləri qaytarır.

Yekun slice() metodu:

  • Massivlərdən və stringlərdən hissə götürmək üçün daha təhlükəsiz və çevik bir üsuldur.
  • Orijinal məlumat toxunulmaz qalır.
  • Müxtəlif praktiki hallarda istifadə edilə bilər: massivləri kopyalamaq, xüsusi hissələri seçmək, mənfi indekslərlə arxadan hissələri çıxarmaq.
