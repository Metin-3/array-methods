sort() array method

JavaScript-də sort() metodu, array-dəki elementləri müəyyən qaydada sıralamaq üçün istifadə edilir. Bu metod array-i yerində dəyişdirir (yəni orijinal array üzərində əməliyyat aparır) və sıralanmış array qaytarır.

Əsas Xüsusiyyətlər

  • Standart Davranış:
    • Əgər sort() metoduna heç bir funksiya ötürülməzsə, elementləri string olaraq (Unicode sırasına əsasən) sıralayır.
    • Bu, rəqəmləri düzgün sıralamaya bilər.
  • Custom Sıralama:
    • Öz sıralama qaydanızı müəyyən etmək üçün compareFunction adlanan funksiya ötürə bilərsiniz.
    • Bu funksiya array-dəki elementlər arasında sıralama qaydasını müəyyən edir.

Sintaksis: array.sort(compareFunction)

  • compareFunction (opsional): Sıralama qaydasını təyin edən funksiya. Bu funksiya iki arqument qəbul edir:
    • a: İndiki element.
    • b: Növbəti element.
  • compareFunction qaytarır:
    • negative value (a - b < 0) → a əvvəl gəlir.
    • positive value (a - b > 0) → b əvvəl gəlir.
    • 0 → sıralama dəyişmir.

Əsas Məqamlar

  • Orijinal array dəyişir: sort() metodu array-in özünü dəyişdirir.
  • Müqayisə funksiyasının təbiəti:
    • Əgər compareFunction qaytarmazsa, default string sıralaması tətbiq olunur.
    • Müqayisə funksiyası qaydalar təyin etmək üçün vacibdir.
  • String və rəqəmlərdə fərqlilik:
    • Stringlərdə Unicode sırası istifadə edilir.
    • Rəqəmləri düzgün sıralamaq üçün compareFunction tələb olunur.

Qısa Yekun

  • sort(), array-dəki elementləri sıralamaq üçün istifadə edilir və array-in özünü dəyişir.
  • Stringlər üçün: Default əlifba sırası və Unicode qaydası tətbiq olunur.
  • Rəqəmlər üçün: Düzgün sıralama üçün compareFunction tələb olunur.
  • Custom qaydalar: compareFunction ilə istənilən qayda təyin edilə bilər.
